# Easy Eats

Easy Eats is a simple Android application that helps users explore and view different recipes. The app provides an easy-to-use interface where users can browse recipes and see ingredients along with cooking instructions.

## Features
- Browse different recipes  
- View ingredients and steps  
- Clean and simple user interface  

## Tech Stack
- Kotlin / Java  
- XML / Jetpack Compose  
- Android Studio  

## How to Run
1. Clone the repository  
2. Open the project in Android Studio  
3. Sync Gradle  
4. Run the app on an emulator or a real device  

## Purpose
This project is created for learning Android development and building a basic recipe application.
